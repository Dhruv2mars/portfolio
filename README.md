Frontend-only portfolio built with Next.js (App Router), TypeScript, Tailwind, and Bun. UI structure compatible with shadcn/ui.

## Develop

1. Install deps (Bun): `bun install`
2. Run dev server: `bun run dev`

## Build

`bun run build && bun run start`

## Deploy

Deploy to Vercel. The project is static and requires no environment variables.
